学习路线
发表于2022-09-16|学习笔记PythonPython爬虫
爬虫其实是个杂学，涉及的知识点比较广，不仅仅是 Python 的范畴了，学习路线是以下： 先打好 Python 基础，至少学到函数、文件读写、异常处理 前端知识，包括 html、js（如果是爬静态网站，那 js 不怎么重要，了解一下即可，如果是爬动态网站，js 一定得会，不然爬不到数据） 一点网络基础，包括请求体、响应体（爬虫相关教程都会讲这个东西） 之后就是学习 Python 第三方库的使用，爬一些中小网站用 requests 库，会用到上面 123 所有知识点 如果数据量比较大（1w 条以上），那最好学一下数据库存储，sqlite、mysql、mongodb 随便哪一个都可以 到后期爬大型网站用 Scrapy 框架（如果用到 Scrapy 框架，那 Python 至少要学到类与对象）
